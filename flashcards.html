Cómo añadir tarjetas

Solo tienes que duplicar un objeto dentro de flashcards-data.json siguiendo esta estructura:

{ "id": "bloque2-14", "bloque": "Bloque 2", "sesion": 8, "categoria": "Area", "pregunta": "¿Pregunta nueva con formula $A=b \\cdot h$?", "respuesta": "Respuesta nueva con LaTeX: $A=l^2$", "pista": "Pista opcional", "etiquetas": ["area", "repaso"] }
  • id debe ser único.
  • bloque, sesion y categoria alimentan los filtros.
  • etiquetas permite organizar mejor el repaso.
